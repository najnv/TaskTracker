# Task Tracker
Учебное веб-приложения для управления задачами.

## Технологии
- Java 21;
- SpringBoot;
- Spring Data JPA;
- SQLite;
- HTML / CSS / JavaScript

## Запуск
1. Склонировать репозиторий
2. Открыть проект в Intellij IDEA
3. Запустить Spring Boot приложение
4. Открыть http://localhost:8080/index.html
